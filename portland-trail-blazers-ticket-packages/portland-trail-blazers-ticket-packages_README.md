# Portland Trail Blazers: Multi-Game Ticket Package Design

## Case
By 2005 the Trail Blazers faced declining on-court performance, an arena in bankruptcy, and falling attendance and fan sentiment. The case asks how the team should design multi-game ticket packages that fans will prefer, using conjoint survey data on price, seat location, number of games, and promotion type, while staying profitable under the arena's cost constraints.

## Objective
Use conjoint analysis to identify which ticket package attributes matter most to fans, then translate fan preference into a package design that is both attractive to buy and financially viable.

## Approach
Ran a conjoint analysis on attribute importance (games, location, price, promotion), then evaluated candidate packages on two dimensions: total utility (fan preference) and total profit, since the analysis showed the two do not always align.

## Key Findings
- Seat location (39.5%) and price (37.6%) drive the large majority of fan preference; number of games (11.8%) and promotion type (11.2%) matter far less.
- The package with the highest fan utility (200-level midcourt seats, $35, playoff promotion) actually loses $30 per package sold, since 200-level seats carry the highest fixed cost.
- The most profitable package (300-level corner seats, $35, food promotion) generates $118.50 in profit per package while still holding a positive utility score.
- A lower-priced 300-level midcourt package ($25, food promotion) generates a smaller profit ($22.50) but appeals to more price-sensitive fans.

## Recommendation
Offer a flagship 6-game package at $35 for 300-level corner seats with a food promotion as the primary offering, and a secondary 6-game package at $25 for 300-level midcourt seats with the same promotion to capture price-sensitive fans. Avoid any package involving 200-level midcourt seats unless priced at $60 or above, since those combinations are financially unsustainable regardless of their fan appeal.
